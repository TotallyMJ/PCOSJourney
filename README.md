# 🌸 Bloom — PCOS Wellness Journal

แอปบันทึกประจำวัน + ติดตามรอบเดือน + medical tracking สำหรับคน PCOS โดยเฉพาะ
ออกแบบให้ใช้ง่าย ไม่กดดัน ไม่ซีเรียส และเข้าใจร่างกายตัวเองได้มากขึ้น

## ✨ Features

### 🏠 หน้าแรก (Dashboard)
- Quick log 30 วินาที (mood / energy / pain / period)
- Doctor prep banner (auto เมื่อนัดใกล้ถึง)
- รอบปัจจุบัน + คาดการณ์รอบถัดไป + phase tips
- ภาพรวมสุขภาพสัปดาห์: น้ำหนัก · BMI · เป้า · mood เฉลี่ย · นอน · protein
- Awareness alerts + insights + encouragement

### 🌸 บันทึกวันนี้ (Daily — Accordion)
- 13 modules แบบ tap-first (กรอกได้ใน 2 นาที)
- Period flow · symptoms (พร้อม severity) · cravings · pain · mood · energy · sleep · weight · meals · water · exercise · meds · journal
- แตะที่ headcard เพื่อขยาย/ปิด · มี summary ในหัวการ์ดทุกอัน
- **ดูย้อนหลังได้** ผ่าน date navigator (← / picker / →)
- Auto-save ทุกการเปลี่ยนแปลง

### 📊 รายสัปดาห์
- Heatmap 7 วัน × 4 metrics
- Symptom frequency
- Exercise log + Meds adherence

### 📅 รายเดือน
- 🎯 3 takeaways (auto: ดีขึ้น/ต้องสังเกต/ทำได้ดี)
- 📅 Calendar heatmap (แตะวันเพื่อย้อนดู)
- 📊 ภาพรวม + เทียบเดือนก่อน
- 🌟 PCOS Cycle Regularity Score (A/B/C/D)
- 🔬 อาการแยกตามช่วงรอบเดือน
- 🥗 Food correlation auto
- 💊 Meds adherence เป็น day-dot grid (เห็นทุกวัน)
- 📈 Trends 6 เดือน (energy / symptoms / cycle / weight)

### 🩺 ดูแลสุขภาพ (Cycle + Medical)
- ปฏิทินรอบเดือน + 3-month overview
- 🏥 นัดหมอ (พร้อม outcome notes)
- 💉 ผลแล็ป (auto-flag จาก ref range + trend detection)
- 📝 บันทึกการแพทย์ (คำถาม/ข้อสังเกต)
- 📋 **Doctor prep view** (printable PDF — เอาไปให้หมอดู)

### ⚙️ ตั้งค่า
- Profile (DOB, ส่วนสูง, น้ำหนัก, น้ำหนักเป้าหมาย)
- เป้าหมายสุขภาพ (ลดไขมัน / รักษา / เพิ่มกล้าม) → คำนวณ TDEE + protein
- รอบเดือน · รอบตัว (วัดเดือนละครั้ง)
- ยา/supplement (ตั้งเวลา + ก่อน/หลังอาหาร)
- Module reorder (drag-drop)
- โหลดข้อมูลตัวอย่าง 60 วัน

## 🚀 Run

ดับเบิลคลิกเปิด `bloom_app.html` ในเบราว์เซอร์ได้เลย — ไม่ต้องลงอะไร

ข้อมูลทั้งหมดเก็บใน `localStorage` ของเบราว์เซอร์ · ไม่มีการส่งออกไปที่ไหน

## 🔒 Privacy

- 100% local (no server, no analytics, no tracking)
- Export ข้อมูลเป็น JSON ได้ทุกเมื่อในหน้าตั้งค่า
- ลบทั้งหมดได้ด้วยการกด "🗑️ ล้างข้อมูล"
